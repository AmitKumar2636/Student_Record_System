\# What’s the project?



This repository contains code for a student management system written in Python. The Student Records System is a command-line application that a school or training centre can use to manage a class of students. It can store a student’s name, score, grade, attendance history, and teacher notes; read that information back from files on disk; analyze scores to produce class-wide reports; identify who was absent from a session; log an honor roll to a persistent file; update a student’s score and recalculate their grade immediately; print a formatted report card; and run an interactive menu so a teacher can perform all of these tasks without modifying the code.



This will be a CLI app first, and eventually it will be converted into a Django app.
